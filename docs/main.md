# hairtail 带鱼

## 概述
### 为什么
对我来说，决策是一件很困难的事情。我总是不能很好的确定什么是我最应该关注的。同时，我也很难注意到那些更紧急的需要被完成的事情。
我需要一个外部的工具来帮助我做决定，提醒我完成必要的事情。
并且如果条件充足，我还希望他可以提前预警出那些危险的情况，比如是否最近我的状态不佳应该减少心智负担重的任务。
### 怎么做
#### 数据可视化
有人说大脑天生喜欢处理图像信息，无论是否是科学的结论，至少对我来说确实如此。
将所有关心的信息都放在一张图上，可以减少思考的负担。
#### AI整理数据
我讨厌表单。
但是数据展示的基础是固定的数据结构，如果不使用表单填写的方法，该怎么保证数据的完整性？
至少现在有了一个选择：AI。
#### 数据存储
我用飞书记录各种信息，这样做了半年多。飞书的表格有上限，所以我拆分为各种模块分别记录。但是使用数据库呢？
我统计了每个表的字段设计，然后意识到我只在记录两种信息：有时间的活动、和没时间的信息。
他们都有一些共有的字段，但是也总有一些特殊的家伙。我不想为了每一个新出现的特殊的家伙改动代码，那就是你了MongoDB。
### 做什么
#### 整体分析图表
一个可以展示更多信息的日程图。
#### 对话式输入
像聊天一样记录。
#### 每日关注点
专注于当天。
